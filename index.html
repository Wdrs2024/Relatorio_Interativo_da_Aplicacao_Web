Visão Geral do Projeto

O objetivo foi desenvolver uma aplicação web completa, com frontend e backend desacoplados para garantir escalabilidade e manutenção.

Módulo de Contato

Permite que usuários enviem mensagens através de um formulário, que são capturadas e armazenadas no banco de dados.

Módulo de Clientes

Habilita o cadastro de novos clientes, armazenando informações essenciais como nome, e-mail e telefone para gestão.

Módulo de Produtos

Permite a visualização de produtos cadastrados, com possibilidade de futuras expansões para gerenciamento de estoque.

Tecnologias Utilizadas

Um stack de tecnologias modernas foi escolhido para garantir um desenvolvimento robusto e eficiente.

Linguagens e Ferramentas

  • Node.js: Ambiente de execução para JavaScript no servidor.
  • Vue.js: Framework para construção de interfaces de usuário.
  • TypeScript: Linguagem com tipagem estática utilizada no Frontend.
  • PostgreSQL: Banco de dados relacional para persistência de dados.

Frameworks e Bibliotecas

  • Express.js: Framework para a criação da API RESTful em Node.js.
  • Axios: Cliente HTTP para comunicação com a API.
  • Vue Router: Biblioteca para roteamento no frontend.

Estilização e Deploy

  • Tailwind CSS: Framework de CSS utility-first para estilização rápida.
  • Vercel: Plataforma de deploy para o Frontend.
  • Render: Plataforma de deploy para o Backend e DB.

Fluxos da Aplicação: Formulário de Contato

Visualize o fluxo de dados do formulário de contato, desde a interação do usuário até o armazenamento no banco de dados.

1. Usuário

Preenche e envia o formulário de contato na interface.

→
2. Frontend (Vue.js)

Aciona o método `enviarFormulario()` e faz uma requisição POST com Axios.

→
3. Backend (API)

A rota `/api/contacts` no **Render** recebe os dados da requisição.

→
4. Banco de Dados (PostgreSQL)

A API executa uma query SQL para inserir os dados na tabela `contacts`.

→
5. Resposta

A API retorna um status de sucesso (200/201) e o frontend exibe uma mensagem de confirmação.
